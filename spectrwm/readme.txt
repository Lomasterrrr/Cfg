Открыть терминал			win + enter		его бинарник должен быть по пути ~/st/st
Открыть меню (sout)			win + d			его по пути ~/sout/sout
Открыть Telegram			win + m			его по пути ~/Telegram/Telegram
Сделать скриншот			win + z			скрипт для этого по пути ~/scripts/sc.sh
Менять режимы окон			win + space
Скрыть бар				win + b
Перейти на следующее окно		win + j
Перейти на предыдущие окно		win + k
Закрыть окно				win + q
Расширять окно				win + h
Уменьшать окно				win + l
Сделать окно fullscreen			win + e
Сделать окно реально fullscreen		win + shift + e
Перезагрузить spectrwm			win + shift + r
Включить/выключить на окне float mode	win + t
Поменять окна местами			win + shift + \
Перейти на рабочий стол номер <n>	win + <n>		всего их может быть 10, десятый имеет номер 0
Переместить текущ. окно на стол <n>	win + shift + <n>
Выйти из spectrwm			win + shift + q
